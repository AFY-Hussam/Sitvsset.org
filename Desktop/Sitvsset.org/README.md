## Run it

This is a **no-build** website (plain HTML/CSS/JS).

- Open `index.html` in a browser, or
- Use any local server (recommended) so audio + canvas behave consistently.

## Controls

- **Speed Quiz**: click **SIT** / **SET** (keyboard: **S** = sit, **D** = set)
- **Drag Battle**: drag phrase cards into **SIT** or **SET**
- **Boss Fight**: click **SIT** / **SET** (keyboard: **J** = sit, **K** = set)
- **Audio toggle**: top-right button
